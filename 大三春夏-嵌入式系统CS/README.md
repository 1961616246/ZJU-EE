## 课程介绍

这门课是CS的嵌入式系统课程。放在CS，这课就是门硬件课了，因为涉及到GPIO、定时器等；但放在EE，这真的算是门软件课，相比于自动化的嵌入式系统，代码量还是要多不少的。这门课是春夏长学期的课，春学期基本就是stm32应用，夏学期会涉及一些嵌入式操作系统，嵌入式linux内核模块编写等内容。
